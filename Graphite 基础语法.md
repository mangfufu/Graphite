## 列表

### 无序列表

-   项目 A
    
-   项目 B
    
    -   子项目 B1
        

```
-+空格+项目 A
-+空格+项目 B
  -+空格+子项目 B1
```

### 有序列表

1.  第一步
    
2.  第二步
    

```
1.+空格+第一步
2.+空格+第二步
```

### 任务列表

-   已完成
    
-   未完成
    

```
-+空格+[x]+空格+已完成
-+空格+[ ]+空格+未完成
```

-   \[\] s d
