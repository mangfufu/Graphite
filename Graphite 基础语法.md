# Graphite 基础语法

Graphite 是所见即所得编辑器。输入以下 markdown 语法会实时渲染，也可以使用工具栏按钮。

---

## 标题

行首输入 `#` 后按空格：

```
# 标题 1
## 标题 2
### 标题 3
#### 标题 4
##### 标题 5
###### 标题 6
```

## 文本样式

```
**粗体**
*斜体*
***粗斜体***
~~删除线~~
<u>下划线</u>
==高亮==
`行内代码`
```

## 列表

### 无序列表

行首输入 `- ` 或 `* `：

```
- 项目 A
- 项目 B
  - 子项目 B1
```

### 有序列表

行首输入 `1. `：

```
1. 第一步
2. 第二步
```

### 任务列表

```
- [x] 已完成
- [ ] 未完成
  - [ ] 嵌套任务
```

## 引用

行首输入 `> `：

```
> 引用文本
> > 嵌套引用
```

## 代码块

输入 ` ``` ` 后按回车：

````
```javascript
function hello() {
  console.log("Hello")
}
```
````

支持高亮的语言：javascript、typescript、rust、python、html、css、json、bash 等。

## 链接与图片

```
[链接文字](https://example.com)

![图片描述](图片路径.png)
```

图片支持 png、jpg、jpeg、gif、bmp、webp、svg，通过工具栏 🖼 插入。

## 表格

```
| 左对齐 | 居中 | 右对齐 |
| :----- | :--: | ----: |
| 单元格 | 单元格 | 单元格 |
```

## 分割线

```
---
```

## 定义列表

```
术语
: 定义内容
: 另一个定义
```

## 脚注

```
正文文字[^1]

[^1]: 脚注内容
```

点击 `[1]` 跳到定义，点击定义中的 `[1]` 跳回正文。

## 快捷键

| 快捷键 | 功能 |
|--------|------|
| Ctrl+S | 保存 |
| Ctrl+O | 打开文件夹 |
| Ctrl+F | 搜索 |
| Ctrl+B | 粗体 |
| Ctrl+I | 斜体 |
| Ctrl+Z | 撤销 |
| Ctrl+Shift+Z | 重做 |
| Ctrl+\ | 切换侧边栏 |
| Ctrl+Shift+P | 命令面板 |
| F11 | 全屏 |
