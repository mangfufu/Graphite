# Graphite 基础语法

本文档列出 Graphite 支持的所有 Markdown 语法。可以直接复制到编辑器中查看渲染效果。

---

## 标题

```
# 标题 1
## 标题 2
### 标题 3
#### 标题 4
##### 标题 5
###### 标题 6
```

## 文本样式

```
**粗体**
*斜体*
***粗斜体***
~~删除线~~
<u>下划线</u>
==高亮==
`行内代码`
```

## 列表

### 无序列表

```
- 项目 A
- 项目 B
  - 子项目 B1
  - 子项目 B2
```

### 有序列表

```
1. 第一步
2. 第二步
3. 第三步
```

### 任务列表

```
- [x] 已完成
- [ ] 未完成
  - [ ] 嵌套任务
```

## 链接与图片

```
[链接文字](https://example.com)
![图片描述](图片路径.png)
```

## 引用

```
> 这是一段引用。
> 引用可以跨多行。
> > 引用可以嵌套。
```

## 代码块

````
```javascript
function hello() {
  console.log("Hello!")
}
```
````

支持语法高亮的语言：javascript、typescript、rust、python、html、css、json、bash 等。

## 表格

```
| 左对齐 | 居中对齐 | 右对齐 |
| :----- | :------: | -----: |
| 单元格 |  单元格  |  单元格 |
```

## 分割线

```
---
```

## 文本对齐

使用工具栏的对齐按钮：左对齐、居中、右对齐。

## 定义列表

```
术语
: 定义内容
: 另一个定义

另一个术语
: 对应的定义
```

## 脚注

```
正文文字[^1]

[^1]: 脚注内容
```

- 点击正文中的 `[1]` 跳转到脚注定义
- 点击脚注定义中的 `[1]` 跳转回正文

## 图片

工具栏点击 🖼 选择本地图片，支持 png、jpg、jpeg、gif、bmp、webp、svg 格式。
